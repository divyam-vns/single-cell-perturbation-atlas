# Single-Cell Perturbation Response Atlas for Disease State Prediction

## Project Overview
This project builds a single-cell perturbation response atlas using real scRNA-seq datasets from cardio-renal-metabolic tissues. The goal is to identify cellular responses to perturbations and develop machine learning models that predict disease states from single-cell transcriptomic profiles.

## Objectives
- Process and integrate real single-cell datasets
- Characterize perturbation-induced cellular responses
- Identify disease-associated gene programs
- Build predictive disease-state models
- Create a reproducible, portfolio-quality bioinformatics workflow

## Technology Stack
- Python
- Scanpy
- scvi-tools
- AnnData
- Scikit-learn
